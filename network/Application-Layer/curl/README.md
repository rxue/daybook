# Options
## `-i` / `--include`
> include response headers in the output
## `-O` for download
### `--remote-name`
> Write output to a file named as the remote file
## `--netrc` or `-n`
> Makes curl scan the .netrc file in the user's home directory for login name and password. This is typically used for FTP on Unix. If used with HTTP, curl enables user authentication.
